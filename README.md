### :purple_heart: 프로젝트 소개

<img src="https://user-images.githubusercontent.com/54823396/87871935-a886e180-c9ef-11ea-8ed1-8811c111e7d1.png" width="20%">

**We: Diary**는 함께 쓰는 일기장입니다. 원하는 친구와 함께 일기장을 생성하고 일기를 작성할 수 있습니다.
